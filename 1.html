HTML이란 무엇인가??

Hypertext Markup Language (HTML) 웹 페이지 및 웹 응용 프로그램을 만들기위한 표준 마크 업 언어입니다. 웹 브라우저는 웹 서버 또는 로컬 저장소에서 HTML 문서를 수신하여 멀티미디어 웹 페이지로 렌더링합니다. HTML은 웹 페이지의 구조를 의미 론적으로 설명하며 원래 문서 모양에 대한 단서를 포함합니다

HTML 요소는 HTML 페이지의 구성 요소입니다. HTML 구조, 이미지 및 대화 형 양식과 같은 기타 개체를 사용하면 렌더링 된 페이지에 삽입 될 수 있습니다. 제목, 단락, 목록, 링크, 인용문 및 기타 항목과 같은 텍스트의 구조적 의미를 표시하여 구조화 된 문서를 작성하는 수단을 제공합니다. HTML 요소는 꺾쇠 괄호를 사용하여 작성된 태그로 표시됩니다.
